DOM = Document Object Model

É a vertente WEB do JavaScript.
Conjunto de objetos dentro do navegador que dá acesso aos componentes internos do website. http://127.0.0.1:5500//

Árvore DOM

Começa pelo window = root(raíz)
Dentro do vários objetos, o window possui o Document = onde está o html do site
Dentro do Document, está o body e o head (children)
Cada parent pode conter várias childs que por sua vez podem conter inúmeros métodos

5 métodos de acesso da Árvore DOM

Por Marca (tag)

É feito utilizando o comando getElementsByTagName('tag')[índice] -> o índice começa do 0.

Esse é um parágrafo de teste, para demonstrar as funcionalidades do DOM.

Por ID

Tag div usando ID para tastar o getElementsByID

Utiliza o getElementById("nome_id")

O mesmo princípio de utilização do getElementById funciona para getElementsByName e getElementsByClass
